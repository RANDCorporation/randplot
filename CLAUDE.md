## What This Package Does

**randplot** is an R package providing a ggplot2 theme (`theme_rand()`) and color palettes for RAND Corporation's Communications Design style. It is used to produce consistently styled charts and graphics.

## Common Commands

All development is done via R (ideally in RStudio). Use the `devtools` package for development workflows.

```r
# Load package interactively during development
devtools::load_all()

# Run all tests
devtools::test()

# Run a single test file
testthat::test_file("tests/testthat/test-test_randplot.R")

# Regenerate documentation (NAMESPACE and man/ files)
devtools::document()

# Run R CMD CHECK (full package check)
devtools::check()

# Install the package locally
devtools::install()
```

## Architecture

The package is minimal — three source files in `R/`:

- **`R/rand-pal.R`** — Defines color palettes (`RandCatPal`, `RandGrayPal`) and `show_rand_pal()` for visualizing them.
- **`R/theme-rand.R`** — Defines `theme_rand(font)`, the primary ggplot2 theme. Accepts `"Helvetica"` (web default) or `"Helvetica Neue"` (print).
- **`R/randplot.R`** — Package-level roxygen2 documentation only.

`NAMESPACE` is auto-generated by roxygen2 — do not edit it directly. Run `devtools::document()` to regenerate after changing `@export` tags or roxygen blocks.

## Testing

Tests use testthat (v3+). The test suite lives in `tests/testthat/`. CI runs `R CMD CHECK` across macOS, Windows, and multiple Ubuntu/R versions via `.github/workflows/check-full.yaml`. Coverage is reported to Codecov via `.github/workflows/test-coverage.yaml`.

## Conventions

- All exported symbols are prefixed with `Rand` (PascalCase for data objects) or `rand_` / `theme_rand` (snake_case for functions).
- Examples in roxygen blocks are wrapped in `\dontrun{}` since they require interactive use.
- 2-space indentation, UTF-8 encoding (per `randplot.Rproj`).
